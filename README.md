## 🚀 Quick Start

### Prerequisites
- Node.js 16+ and npm
- Modern web browser
- Git

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Ankit500ak/AirAware.git
   cd AirAware/air-aware-app
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Environment setup**
   ```bash
   # Create .env file in the root directory
   touch .env
   
   # Add your EmailJS credentials to .env
   REACT_APP_EMAILJS_PUBLIC_KEY=your_public_key_here
   REACT_APP_EMAILJS_SERVICE_ID=your_service_id_here
   REACT_APP_EMAILJS_TEMPLATE_ID=your_template_id_here
   ```

4. **Start development server**
   ```bash
   npm start
   ```

5. **Open in browser**
   Navigate to `http://localhost:3000`

### Available Scripts

- `npm start` - Runs the app in development mode
- `npm test` - Launches the test runner
- `npm run build` - Builds the app for production
- `npm run eject` - Ejects from Create React App (one-way operation)

## 🔧 Environment Setup

### Required Environment Variables
Create a `.env` file in the root directory:

```env
# EmailJS Configuration
REACT_APP_EMAILJS_PUBLIC_KEY=your_public_key
REACT_APP_EMAILJS_SERVICE_ID=your_service_id
REACT_APP_EMAILJS_TEMPLATE_ID=your_template_id

# Optional: API Keys for future integrations
REACT_APP_NASA_API_KEY=your_nasa_api_key
REACT_APP_CPCB_API_KEY=your_cpcb_api_key
```

### Getting EmailJS Credentials
1. Visit [EmailJS.com](https://www.emailjs.com/)
2. Create an account and service
3. Set up email template
4. Get your Public Key, Service ID, and Template ID
5. Add them to your `.env` file
